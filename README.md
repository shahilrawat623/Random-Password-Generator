# Random-Password-Generator
Generates strong, customizable random passwords using HTML, CSS, and JavaScript.

A simple, lightweight web app that generates strong, customizable random passwords using **HTML, CSS, and JavaScript**.

---

## 🚀 Features
- Generate random passwords instantly
- Choose password length
- Include/exclude uppercase letters, numbers, and symbols
- Copy password to clipboard with one click
- Clean and responsive UI

---

## 📂 Project Structure
index.html   → Main HTML file (UI layout)<br>
style.css    → Styling for the app (colors, fonts, responsiveness)<br>
script.js    → JavaScript logic for password generation

## 🧰 Technologies Used
- HTML
- CSS
- JavaScript
